# Bot Dashboard

Управление Yandex-ботами. Запускается одной командой.

## Требования

- **Node.js >= 18** — [nodejs.org](https://nodejs.org)
- yandex_bot должен лежать в `~/yandex_bot` (или задать `BOT_DIR` в `.env`)

## Запуск

```bash
./start.sh
```

Открыть браузер: **http://localhost:4000**

Порт можно изменить — создать `.env` рядом с `start.sh`:
```
PORT=4000
BOT_DIR=/другой/путь/к/yandex_bot
```

## Что умеет

| Вкладка | Что делает |
|---------|-----------|
| **Боты** | Запуск / остановка / логи, состояние каждого бота |
| **Прокси** | Добавить / заменить / удалить, статус бана |
| **Google Docs** | Ссылки на документы с запросами |
| **Автопилот** | Автозапуск ботов по расписанию, авто warmup→target, задержка перезапуска |

## Для разработчика

```bash
pnpm install
pnpm dev        # dev-режим на порту 4000
bash build.sh   # пересобрать release/
```
